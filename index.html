Vyper Repo

iOS Tweak Repo for Jailbroken Devices

Axe

Axe is simple tweak to add changing colors to the battery icon. It goes down from a bright green at full battery to a dark red at low battery. It has a small one switch preference page with a non-respringing change applier.

Install Axe
TransDock

TransDock is another simple tweak with the sole purpose to make the dock transparent. It also has a simple one switch preference page with a quick respring to apply changes. It is quick, simple, and easy to use.

Install TransDock
Cyphus

Cyphus is a more complex tweak, changing the color of the notification badge on each app according to the app's primary icon. No options to configure as well as a simple preference page.

Install Cyphus

My Github profile of personal projects

Brute-Force Password Cracker

This is a proof of concept program I wrote, showing how programs like this work. It takes a user-given password and submits characters until the whole string is solved. In practice, this concept is much more difficult but aims to help people understand the importance of proper passwords.

View Project
Single Layer Text Encryption

This is a single layer encyption program based off of the iPhone keyboard. This type of encryption is meant to be able to be used interchangibly between normal typing and encrypted writing. While meeting this criteria, the program mixes up a few characters but is not crucial.

View Project
p0tion

p0tion was a work-in-progress jailbreak for up to iOS 14.3. The goal of a jailbreak is to allow users to have more control over their devices. I lead a team of about 4 people to try to make this program. But, following other developers' releases, my team's project was haulted and archived.
