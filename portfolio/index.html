POC: Brute-force Password Cracker

This is a non-malicious proof of concept program I wrote. It demonstrates how brute force password crackering work. It takes a user-given password and submits characters until the whole string is solved. In practice, this concept is much more difficult, but the program aims to help people understand the importance of proper passwords/security.

Single Layer Text Encryption

This is a basic type of encryption based off the default iPhone keyboard layout. It takes each letter from a string and changes it to the corrosponding character. This type of encryption is meant to be able to be used interchangibly between normal typing and encrypted writing. Due to this versitality, a few characters are mistaken for another when decrypting, but is not a crucial problem to it's functionality.

p0tion

p0tion was a work-in-progress jailbreak for iOS 14.0-14.3 on any device. The goal of a jailbreak is to escape the "sandbox" allow users to have more control over their devices. I lead a team of about 4 people to try to make this program. But, following other developers' releases, my team's project was haulted and archived. We did however, get pretty close and now the code is open-source for anyone to view.
